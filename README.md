    - 3.Python Variables
    
    다른 유형으로 변수 설정 가능
    
    ```jsx
    x = 5
    y = "John"
    print(x)
    print(y)
    
    ```
    
    변수는 더욱 더 길게 설정할 수 있다.
    
    ```jsx
    myvar = "John"
    my_var = "John"
    _my_var = "John"
    myVar = "John"
    MYVAR = "John"
    myvar2 = "John"
    
    ```
    
    여러 값으로 설정 가능
    
    ```jsx
    x, y, z = "Orange", "Banana", "Cherry"
    print(x)
    print(y)
    print(z)
    
    ```
    
    +쉼표로 연산 가능(수와 문자는 연산 X)
